# YouBike 即時資訊應用程式

此應用程式顯示臺北市 YouBike 站點的即時資訊，並提供篩選、搜尋和排序功能。

## 功能
- 篩選：依據行政區篩選站點資訊
- 搜尋：根據站點名稱搜尋
- 排序：依據可租借車輛數和可歸還空位數進行排序
- 每分鐘自動更新資料

## 安裝與運行

1. clone此專案：
   ```bash
   git clone https://github.com/你的帳號/ubike-info.git
2. 進入專案資料夾並安裝依賴套件：
    cd ubike-tommy
    npm install
3. 運行應用程式：    
    npm start
